Welcome to Calc Pi’s documentation!¶

This is a tutorial example for BLT (https://github.com/llnl/blt) that creates C++ libraries that calculate serially and in parallel using MPI and CUDA.

These libraries calculate by approximating the integral using numerical integration. In the MPI implementation, the intervals are distributed across MPI tasks and a MPI_AllReduce calculates the final result. In the CUDA implementation, the intervals are distributed across CUDA blocks and threads and a tree reduction calculates the final result.

The method is adapted from: https://www.mcs.anl.gov/research/projects/mpi/usingmpi/examples-usingmpi/simplempi/cpi_c.html
